# SOP: Progress Tracking (Отслеживание прогресса)

## Цель
Собирать статистику выполнения задач, формировать вечерний отчёт и отслеживать streak (дни подряд с выполненными задачами).

---

## Входные данные
- `user_id` — UUID пользователя
- `date` — дата отчёта (по умолчанию: сегодня)
- Задачи из Supabase за указанную дату

## Выходные данные
- Вечерний отчёт в Telegram
- Запись в `daily_reports` (тип: `evening_summary`)
- Обновлённый streak в `users`

---

## Бизнес-правила

1. **Вечерний отчёт:** отправляется ежедневно (настраивается через n8n Cron)
2. **Метрики отчёта:**
   - Задач выполнено / всего на день
   - Процент выполнения
   - Общее время (actual_minutes или estimated_minutes)
   - Текущий streak
3. **Streak:** считается по дням подряд, где ≥ 1 задача завершена
   - Если за день 0 выполненных задач → streak сбрасывается
   - Если задач не было запланировано → streak НЕ сбрасывается (выходной)
4. **Мотивация:** при streak ≥ 3 дня добавлять 🔥, при ≥ 7 — 🏆
5. **Completion rate:** tasks_done / tasks_total * 100

---

## Формат вечернего отчёта (Telegram Markdown)

```
🌙 *Итоги дня 2026-04-06*

✅ Выполнено: 4/5 (80%)
⏱ Время: ~2ч 15мин
🔥 Streak: 5 дней подряд!

Выполненные:
✅ Написать функцию авторизации
✅ Исправить баг в checkout
✅ Изучить документацию Stripe API
✅ Обновить README проекта

Не выполненные:
❌ Ответить клиенту → перенесено на завтра

━━━━━━━━━━━━━━━
💪 Отличная работа! Продолжай в том же духе!
```

## Edge Cases

| Ситуация | Поведение |
|----------|-----------|
| Все задачи выполнены (100%) | Поздравительное сообщение 🎉 |
| 0 задач выполнено | Мягкое напоминание, без осуждения |
| Не было задач на день | «Сегодня был выходной. Отдохни! 😴» |
| Есть `skipped` задачи | Показать отдельно, не считать как невыполненные |
| Supabase недоступен | Логировать, отложить отчёт |
